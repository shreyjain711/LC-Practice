### ***[Trapping Rain Water](https://leetcode.com/problems/trapping-rain-water/)***:
- ***Problem Desc***: given a list of heights of towers at position i, find out how much water will be retained between them
- ***find out maxLeft and maxRight before hand, for each pos we know left and right max wall size and water retention capacity, add it up [O(n) time | O(n) space]***:
  ```cpp
  vector<int> maxSide(int i, int d, int n, vector<int>& height) {
      int m = 0; vector<int> ans(n, 0);
      while(i<n && i>=0) {
          ans[i] = m; m = max(m, height[i]);
          i+=d;
      } return ans;
  }
  int trap(vector<int>& height) {
      int n=height.size(), ans = 0; 
      vector<int> maxL = maxSide(0, 1, n, height);
      vector<int> maxR = maxSide(n-1, -1, n, height);
      for (int i=0; i<n; i++) {
          ans += max(0, min(maxL[i], maxR[i]) - height[i]);
      } return ans;
  }
  ```
- ***Left and right pointer, increment the smaller one, maintain the maxLeft and maxRight vals and find water retained at the moving pos [O(n) time | O(1) space]***:
  ```cpp
  int trap(vector<int>& height) {
      int n=height.size(), ans = 0, i=0, j=n-1, maxL=0, maxR=0; 
      while (i<j) {
          maxL = max(maxL, height[i]);
          maxR = max(maxR, height[j]);
          ans += max(0, min(maxL, maxR)-min(height[i], height[j]));
          if (height[i]>height[j]) j--;
          else i++;
      } return ans;
  }
  ```

***[Construct Binary Tree From Preorder And Inorder Traversal](https://leetcode.com/problems/construct-binary-tree-from-preorder-and-inorder-traversal/)***
- ***Problem Desc***: given inorder and preorder of the tree, reconstruct it
- ***maintain reference for preorder pointer, maintain range for inorder and create till il<=ir and pi<n [O(n) time avg (can go O(n^2) | O(1) space]***:
  ```cpp
  TreeNode* recurHelper(int &pi, int il, int ir, vector<int>& preorder, vector<int>& inorder) {
      if (il>ir || pi>=preorder.size()) return nullptr;
      TreeNode* root = new TreeNode(preorder[pi]);
      int i=il;
      for (; i<inorder.size(); i++) {
          if (inorder[i]==preorder[pi]) break;
      } pi++;
      root->left = recurHelper(pi, il, i-1, preorder, inorder);
      root->right =recurHelper(pi, i+1, ir, preorder, inorder);
      return root;
  }
  TreeNode* buildTree(vector<int>& preorder, vector<int>& inorder) {
      int pi=0;
      return recurHelper(pi, 0, inorder.size(), preorder, inorder);
  }
  ```
