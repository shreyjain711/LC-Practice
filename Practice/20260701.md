### ***[Longest Consecutive Sequence](https://leetcode.com/problems/longest-consecutive-sequence/)***:
- ***Problem Desc***: given list of nums, find longest consec sequence. not sorted, duplicates exist
- ***make hashset, iterate on elements, check if it's a start (previous doesn't exist), count, track it's evaluated for duplicates [O(n) time | O(n) space]***:
  ```cpp
  int longestConsecutive(vector<int>& nums) {
      int ans = 0;
      unordered_set<int> s(nums.begin(), nums.end()), tried;
      for (int i=0; i<nums.size(); i++) {
          if (s.count(nums[i]-1) || tried.count(nums[i])) continue;
          int curr = nums[i], cnt = 0; tried.insert(curr);
          while (s.count(curr++)) cnt++;
          ans = max(ans, cnt);
      } return ans;
  }
  ```
- ***make hashset, iterate on elements, check sequence both ways, remove from set [O(n) time | O(n) space]***:
  ```cpp
  int longestConsecutive_bothways(vector<int>& nums) {
      unordered_set<int> s(nums.begin(), nums.end());
      int ans = 0;
      for (int i=0; i<nums.size(); i++) {
          if (s.find(nums[i]) != s.end()) {
              int curr=nums[i], lcs = 1;
              while (s.find(--curr) != s.end()) {
                  lcs++; s.erase(curr);
              } curr = nums[i];
              while (s.find(++curr) != s.end()) {
                  lcs++; s.erase(curr);
              } ans = max(ans, lcs);
          }
      }
      return ans;
  }
  ```

***[3Sum](https://leetcode.com/problems/3sum/)***:
- ***Problem Desc***: given inorder and preorder of the tree, reconstruct it
- ***Sort, two pointer for rest; move pointers and check they're not the same as prev pos [O(n.logn) time | O(n) space]***:
  ```cpp
  vector<vector<int>> threeSum(vector<int>& nums) {
      vector<vector<int>> ans;
      int ii = 0, n = nums.size();
      sort(nums.begin(), nums.end()); 
      if (nums[n-1]<0) return ans;
      while (ii<n && nums[ii]<=0) {
          int i = ii+1, j = n-1; 
          while (i<j) {
              int csum=nums[ii]+nums[i]+nums[j];
              bool mvi = csum<0, mvj = csum>0, mvb = !csum;
              if (mvb) ans.push_back({nums[ii], nums[i], nums[j]});
              if (mvi||mvb) {i++; while(i<n && nums[i-1]==nums[i]) i++;}
              if (mvj||mvb) {j--; while(j>=0 && nums[j+1]==nums[j]) j--;}
          } ii++; while (ii<n && nums[ii] == nums[ii-1]) ii++;
      } return ans;
  }
  ```
